# Timeplus MCP Demo Docker Stack

This example contains a demo Docker stack for Timeplus MCP.


## Prerequisites

Before you begin, make sure you have the following prerequisites:

- Docker installed on your machine


## Getting Started

just run `docker compose up` and then go to browser `localhost:8000` which will run the onboarding process for the first time use.

the follow environment config the MCP on port 3001.

```
MCP_ENABLED=true
MCP_TRANSPORT=http
MCP_PORT=3001
```
